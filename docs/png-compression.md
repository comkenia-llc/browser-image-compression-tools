# PNG Compression

PNG is useful for screenshots, logos, icons, transparent graphics, UI images and sharp text.

PNG files can be larger than JPG because PNG keeps sharp details and can support transparency. If the image is a photo and transparency is not needed, JPG or WebP may create a smaller file.

## When to Use PNG

Use PNG for:

- Screenshots
- Logos
- Icons
- Transparent images
- UI graphics
- Document screenshots
- Images with sharp text

## Best PNG Compression Settings

A good starting point is:

- Resize very large PNG images
- Keep PNG when transparency is needed
- Convert to WebP when smaller website images are required
- Convert to JPG when the image is a normal photo
