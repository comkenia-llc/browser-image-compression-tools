# Target KB Image Compression

Target KB image compression means reducing an image until it reaches a specific file size, such as 20KB, 50KB, 100KB, 200KB or 500KB.

This is useful when a website or online form says your image must be under a fixed upload limit.

## Common Target Sizes

Common image upload limits include:

- Under 20KB
- Under 50KB
- Under 100KB
- Under 200KB
- Under 500KB

## Where Target KB Compression Is Useful

Target KB compression is useful for:

- Online forms
- Exam portals
- Government websites
- Job applications
- Visa forms
- Profile photos
- Document uploads
- Email attachments

## How Target KB Compression Works

A browser image compressor usually tries to reduce image size by:

1. Resizing very large dimensions
2. Reducing image quality
3. Exporting the image again
4. Checking the new file size
5. Repeating until the image is near the target KB size
